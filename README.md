# alexa-sail
Asistente virtual tipo alexa en python. Se límita a 8 posibles comandos de:
- Busqueda en Google
- Busqueda en Youtube
- Decir día
- Decir hora
- Despedirse
- Busqueda en Wikipedia
- Saludar
- Precio del dólar a la tasa del BCV
